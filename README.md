# 技术栈

基于 Vue3 + Vite4 + TypeScript + Pinia + Antd4 的后台系统模板

## 安装依赖

```shell
pnpm install
```

## 启动项目

```shell
pnpm run dev
```

## 打包项目

```shell
pnpm run build
```

## 代码提交

```shell
pnpm run add-commit
```
